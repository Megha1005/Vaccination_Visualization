Exploratory data analysis using visualization on the Covid-19 vaccination data set to understand vaccination trends in different countries.

Target Data Set:

Kaggle - Covid-19 vaccination dataset.

  • What is the progress of vaccination in different countries?
  • Find top countries leading in the vaccination program
  • Count of Fully Vaccinated people in different countries
  • Total Vaccinated per hundred

The target uses can be anyone interested in understanding the vaccination trends in different countries right from health-care professionals to data-scientists.
